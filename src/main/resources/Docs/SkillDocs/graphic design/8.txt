From Wikipedia, the free encyclopedia Jump to: navigation, search This is a category of articles relating to software which can be freely used, copied, studied, modified, and redistributed by everyone that obtains a copy: “free software” or “open source software”. Typically, this means software which is distributed with a free software license, and whose source code is available to anyone who receives a copy of the software. Free software portal Wikimedia Commons has media related to Free graphics software. External links[edit] Graphics Planet is a planet website of people involved in free software development in the field of computer graphics and publishing.
