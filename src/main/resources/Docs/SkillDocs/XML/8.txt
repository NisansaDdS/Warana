Contents 1 See also 2 NOTE 3 References 4 External links See also[edit] Integrated development environment XML XForms NOTE[edit] Spring Tools Suite and Eclipse have a variety of built-in and free plugins matching or exceeding the quality of the paid versions. They come with a content assist tool that completes tags and can search for classes in any java classpath. They also include validation, bean creation, and commit tools. A plugin was also developed for Notepad++ named XML Tools that contains many features including manual/automatic validation, xpath evaluation, auto-completion, pretty print, and text conversion in addition to being able to work on multiple files at once. Other tools are available to run xhtml.
